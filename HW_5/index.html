Види селекторів

  1. Универсальні селектори
  2. Селектор по тегу
  3. Селектор по id

Универсальний селектор "*" - це за його допомогою можна обнулити відступи та задати значення box-size для всіх блочних елементів.

Селектор по тегу ( div, h1-h6, p ) - це селектор CSS застосовує стилі до всіх елементів з однаковим тегом.

Будуть обрані:

  • За назвою елемента
  • За id
  • За класом
  • За кількома класами
  • За вкладеністю
